Principais formatações:

Dica: Ctrl+Shift+P > Emment: Wrap With Abbreviation > tag

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (semântica)

Marcador de Texto

Dica: <mark style="background-color: ..."> para mudar a cor do marcador

Podemos criar também um texto marcado usando a tag <mark>

Texto Grande e Pequeno

Nesta frase temos um texto grande usando a tag <big> (obsoleto)

Nesta frase temos um texto pequeno usando a tag <small> (corrente)

Texto Deletado e Inserido

Textos em excluído são utilizados para indicar que eles devem ser desconsiderados, usa-se a tag <del>

Textos em inserido são utilizados para sublinhar um texto, usa-se a tag <ins>

Texto Sobrescrito e Subscrito

Textos sobrescritos são utilizados, por exemplo, para escrever equações como x2+3, usa-se a tag <sup>

Textos subscritos são utilizados, por exemplo, para escrever fórmulas qúmicas como H2O, usa-se a tag <sub>
